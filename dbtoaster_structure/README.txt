
Files pertaining to the reference DBToaster structure:  code for materialization and maintenance for the Classic IVM and DBToaster IVM options

The code for the other 3 options (naive search, set-based index, and TreeToaster IVM) are contained in the treetoaster_structure files.

The toaster_maintenance.sql file contains the SQL input code used by DBToaster to generate the header files classic_maintenance.hpp and toaster_maintenance.hpp.
